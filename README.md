# OneBlood - Blood Donation Platform

A complete blood donation & emergency request platform for uneducated and rural users, with advanced UI, AI features, 3D visuals, and impactful design.

## Features

- **Ultra-simple blood request**: One large red button for emergency blood requests
- **Real-time hospital dashboard**: Live request monitoring with map integration
- **Donor QR/NFC ID cards**: Fast donation identification with QR codes
- **Multilingual support**: English + Hindi with voice feedback
- **AI-powered matching**: Intelligent donor-recipient pairing
- **Gamification**: Leaderboards, badges, and donor recognition
- **Offline-first**: Queue requests when internet is down
- **PWA capabilities**: Mobile app experience

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite, React Three Fiber, Framer Motion
- **Backend**: Node.js, Express, MongoDB, Socket.io
- **Real-time**: Socket.io for live notifications
- **Maps**: Leaflet + OpenStreetMap
- **Authentication**: JWT tokens
- **Voice**: Web Speech API + pre-recorded audio
- **3D Animations**: React Three Fiber
- **PWA**: Service Worker, Offline capabilities
